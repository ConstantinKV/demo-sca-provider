# demo-sca-provider
Demo SCA Provider client of Salt Edge SCA Service
  
## Build
Run `buildJar` task
 
## Run
Put `application.properties` near jar file, if want to start with custom properties.  

```
java -jar demo-sca-provider-x.x.x.jar
```
